Ventajas

  1. Lenguaje totalmente libre y abierto.
  2. Posee una curva de aprendizaje muy baja.
  3. Los entornos de desarrollo son de rápida y fácil configuración.
  4. Fácil de instalar: existen paquetes autoinstalables que integran PHP rápidamente.
  5. Fácil acceso e integración con las bases de datos.
  6. Posee una comunidad muy grande.
  1. Es el lenguaje con mayor usabilidad en el mundo.
  2. Es un lenguaje multiplataforma.
  3. Completamente orientado al desarrollo de aplicaciones web dinámicas y/o páginas web con acceso a una Base de Datos.
  4. El código escrito en PHP es invisible al navegador ya que se ejecuta al lado del servidor y los resultados en el navegador es HTML.
  5. Posee una versatilidad para la conexión con la mayoría de base de datos que existen en la actualidad.
